# Sentence structure

### **SVTO**

Subject Verb Time Object

Verb always in the second position

### **TEKAMOLO**

| Short form | German   | English | Question                                       |
| ---------- | -------- | ------- | ---------------------------------------------- |
| TE         | temporal | time    | Wann? When?                                    |
| KA         | kausal   | reason  | Warum? Why?                                    |
| MO         | modal    | manner  | Wie? How?                                      |
| LO         | lokal    | place   | Wo? Wohin? Woher? Where? To where? From where? |

**Example**

| Part    | German            | Meaning                    |
| ------- | ----------------- | -------------------------- |
| Subject | Ich               | I                          |
| Verb    | fahre             | travel/go                  |
| **TE**  | morgen            | tomorrow                   |
| **KA**  | wegen des Termins | because of the appointment |
| **MO**  | mit dem Zug       | by train                   |
| **LO**  | nach Salzburg     | to Salzburg                |

In a normal German main sentence, the conjugated verb is in **position 2**.

#### Example 1

**Ich fahre morgen nach Salzburg.**

#### Example 2: time first

You can start with the time, but the verb must still be second:

**Morgen fahre ich nach Salzburg.**

### Perfekt

In Perfekt, the helping verb is in position 2, and the Partizip II goes to the end.

**Ich habe gestern im Supermarkt Brot gekauft.**\
I bought bread yesterday at the supermarket.

### Modal verb

With modal verbs, the modal verb is position 2, and the infinitive goes to the end.

**Ich muss morgen mit dem Bus zur Arbeit fahren.**\
I have to go to work tomorrow by bus.

### Objects: where do Akkusativ and Dativ go?

A common neutral order is:

**Time → Dative object → Accusative object → Manner → Place**

Ich gebe meiner Freundin morgen in der Schule das Buch.

{% hint style="info" %}
Dative person usually comes before accusative thing.
{% endhint %}

### Pronouns change the order

If both objects are nouns:

**Dativ before Akkusativ**

**Ich gebe meiner Schwester das Buch.**\
I give my sister the book.

But if both are pronouns:

**Akkusativ before Dativ**

**Ich gebe es ihr.**\
I give it to her

### Subordinate clauses: verb goes to the end

With words like **weil, dass, wenn, obwohl**, the conjugated verb goes to the end.

#### Main clause

**Ich fahre morgen nach Salzburg.**\
I am travelling to Salzburg tomorrow.

#### With weil

**Ich fahre morgen nach Salzburg, weil ich dort einen Termin habe.**\
I am travelling to Salzburg tomorrow because I have an appointment there.

### Separable verbs: prefix goes to the end

For separable verbs, the prefix goes to the end in present tense.

**Ich stehe um sieben Uhr auf.**\
I get up at seven o’clock.

**Ich kaufe heute im Supermarkt ein.**\
I am shopping today at the supermarket.
