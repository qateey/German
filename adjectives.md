# Adjectives

## Comparative and superlative

| Form       | Meaning          | Example                  |
| ---------- | ---------------- | ------------------------ |
| Grundform  | normal adjective | schnell = fast           |
| Komparativ | more / -er       | schneller = faster       |
| Superlativ | most / -est      | am schnellsten = fastest |

### Comparative

{% hint style="info" %}
adjective + er
{% endhint %}

Mein Auto ist schneller

A ist größer als B.

**größer wie** is common colloquially in some places, but for standard German use **größer als**.

### Superlative

{% hint style="info" %}
am + adjective + sten
{% endhint %}

Mein Auto ist am schnellsten.

### With nouns: use adjective endings

#### Comparative before a noun

**ein schnelleres Auto**\
a faster car

**eine kleinere Wohnung**\
a smaller apartment

**der ältere Mann**\
the older man

#### Superlative before a noun

Use:

**der/die/das + adjective + ste + ending**

der schnellste Zug

das kleinste Zimmer

die besten Freunde/die guten Kinder (Deklination)

### Rules

#### Many one-syllable adjectives get an Umlaut

| Adjective | Comparative | Superlative  |
| --------- | ----------- | ------------ |
| alt       | älter       | am ältesten  |
| jung      | jünger      | am jüngsten  |
| groß      | größer      | am größten   |
| kurz      | kürzer      | am kürzesten |
| warm      | wärmer      | am wärmsten  |
| kalt      | kälter      | am kältesten |
| lang      | länger      | am längsten  |

But not every one-syllable adjective takes Umlaut:

| Adjective | Comparative | Superlative    |
| --------- | ----------- | -------------- |
| schnell   | schneller   | am schnellsten |
| klar      | klarer      | am klarsten    |
| falsch    | falscher    | am falschesten |

#### Adjectives ending in **-d, -t, -s, -ß, -sch, -z, -x** often add **-esten**

This makes pronunciation easier.

| Adjective   | Superlative         |
| ----------- | ------------------- |
| alt         | am ältesten         |
| interessant | am interessantesten |
| laut        | am lautesten        |
| heiß        | am heißesten        |
| frisch      | am frischesten      |
| kurz        | am kürzesten        |

#### Irregular forms

These are very important:

| Basic | Comparative | Superlative |
| ----- | ----------- | ----------- |
| gut   | besser      | am besten   |
| viel  | mehr        | am meisten  |
| gern  | lieber      | am liebsten |
| hoch  | höher       | am höchsten |
| nah   | näher       | am nächsten |

### Equality: use **so ... wie**

For “as ... as”, use:

**so + adjective + wie**

Examples:

**Ich bin so groß wie du.**\
I am as tall as you.

**Diese Tasche ist so teuer wie die andere.**\
This bag is as expensive as the other one.

Compare:

| English     | German      |
| ----------- | ----------- |
| as big as   | so groß wie |
| bigger than | größer als  |
| the biggest | am größten  |
