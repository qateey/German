# Genitiv

## For possession or belonging

In English, it often translates as:

* **of the man**
* **the man’s**

So:

**der Name des Mannes**\
\= the name of the man / the man’s name

For many masculine and neuter nouns in Genitiv, we add **-s** or **-es** to the noun:

| Nominativ | Genitiv    |
| --------- | ---------- |
| der Mann  | des Mannes |
| der Vater | des Vaters |
| das Kind  | des Kindes |
| das Auto  | des Autos  |

**Wessen Name ist Peter?**

Whose name is Peter?

**Der Name des Mannes ist Peter**

### **Sentence structure**

{% hint style="info" %}
der/die/das + thing + des/der + owner
{% endhint %}

| Part           | Role                           |
| -------------- | ------------------------------ |
| **Der Name**   | the thing possessed            |
| **des Mannes** | the owner/person it belongs to |
| **ist Peter**  | is Peter                       |

### Examples

| Question                    | Answer                                                                                                                                           |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Wessen Name ist Peter?**  | Der Name **des Mannes** ist Peter.                                                                                                               |
| **Wessen Auto ist rot?**    | Das Auto **der Frau** ist rot.                                                                                                                   |
| **Wessen Zimmer ist groß?** | Das Zimmer **des Kindes** ist groß.                                                                                                              |
| **Wessen Bücher sind neu?** | Die Bücher **der Kinder** sind neu.                                                                                                              |
| **Wessen Haus ist groß?**   | <p>Das Haus eines mannes ist groß </p><p>Das Haus einer Fraus ist groß </p><p>Das Haus eines Kind ist </p><p>Das Haus seiner Mutter ist groß</p> |
|                             |                                                                                                                                                  |
