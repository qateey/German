# Indefinite pronouns

Words that refer to non-specific people, places, or things,e.g., anyone, anything, somebody

| Word                                                                       | Meaning                 |               |                                                     |
| -------------------------------------------------------------------------- | ----------------------- | ------------- | --------------------------------------------------- |
| <p><strong>alle</strong> </p><p>(changes according to case)</p>            | all / everyone          | people        | alle sind da/ich kenne alle                         |
| <p><strong>viele</strong> </p><p>(changes according to case)</p>           | many                    | people/things | ich helfe vielen/viele kommen heute/ich kenne viele |
| <p><strong>jeder</strong></p><p>(changes according to gender and case)</p> | each / every / everyone | people        | jeder Mann/Jede Frau                                |
| **jemand** (changes according to case)                                     | someone / somebody      | people        | Jemand ist an der Tür.                              |
| **niemand** (changes according to case)                                    | no one / nobody         | people        | Niemand ist da                                      |
| **etwas**                                                                  | something / a little    | things        | ich habe etwas gekauft                              |
| **nichts**                                                                 | nothing                 | things        | ich habe nichts gekauft/Das macht nichts            |
| <p><strong>alles</strong></p><p>(changes according to case)</p>            | everything              | things        | ich brauche alles                                   |
| **manche** (changes according to case)                                     | some people/things      | people/things | manche Bücher/manche Leute lernen schnell           |

| Positive                | Negative             |
| ----------------------- | -------------------- |
| **jemand** = someone    | **niemand** = no one |
| **etwas** = something   | **nichts** = nothing |
| **alle** = everyone/all | **niemand** = nobody |
| **alles** = everything  | **nichts** = nothing |
