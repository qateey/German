# Werden

## To become

|               |       |         |        |
| ------------- | ----- | ------- | ------ |
| **ich**       | werde | **wir** | werden |
| **du**        | wirst | **ihr** | werdet |
| **er/sie/es** | wird  | **sie** | werden |

**Wenn ich viel arbeite, werde ich müde.**

If I work too much, I become tired

**Sie wird Ärztin**

She will become a doctor

**Du wirst immer dünner.**

You are becoming always thinner

## Future tense

{% hint style="info" %}
werden + infinitive at the end
{% endhint %}

**Ich werde Deutsch lernen.**&#x20;

I will learn German

**Du wirst morgen kommen.**&#x20;

You will come tomorrow

## Passive voice

{% hint style="info" %}
werden + Partizip II
{% endhint %}

**Der Brief wird geschrieben.**\
The letter is being written.

**Das Haus wird gebaut.**\
The house is being built.

## Past forms of werden

### Präteritum

|               |         |         |        |
| ------------- | ------- | ------- | ------ |
| **ich**       | wurde   | **wir** | wurden |
| **du**        | wurdest | **ihr** | wurdet |
| **er/sie/es** | wurde   | **sie** | wurden |

**Ich wurde müde.**\
I became tired.

**Sie wurde Ärztin.**\
She became a doctor.

**Das Wetter wurde besser.**\
The weather got better.

### Perfekt

{% hint style="info" %}
**ist + text + geworden**
{% endhint %}

Example:

**Ich bin müde geworden.**\
I became tired / I have become tired.

**Sie ist Ärztin geworden.**\
She has become a doctor.

**Das Wetter ist besser geworden.**\
The weather has gotten better.

Important: **werden** uses **sein** in Perfekt:

**ich bin geworden**, not **ich habe geworden**.

### Passive perfect tense

In the passive perfect tense, German uses **worden**, not **geworden**.

| Meaning                                    | German                            |
| ------------------------------------------ | --------------------------------- |
| The letter is being written.               | Der Brief wird geschrieben.       |
| The letter was written / has been written. | Der Brief ist geschrieben worden. |

## Conditional (Konjunktiv II)

**würde** comes from **werden**, but it means **would**.

{% hint style="info" %}
würde + infinitive at the end
{% endhint %}

|               |         |         |        |
| ------------- | ------- | ------- | ------ |
| **ich**       | würde   | **wir** | würden |
| **du**        | würdest | **ihr** | würdet |
| **er/sie/es** | würde   | **sie** | würden |

**Ich würde gern Deutsch lernen.**\
I would like to learn German.

**Würden Sie mir bitte helfen?**\
Would you please help me?

**Ich würde nach Japan reisen, wenn ich Zeit hätte.**\
I would travel to Japan if I had time.

**Ich würde mehr schlafen.**\
I would sleep more.

## Summary

| **Use of werden** | **Structure**           | **Example**                        |
| ----------------- | ----------------------- | ---------------------------------- |
| become            | werden + adjective/noun | Ich werde müde/Mir wird schwindlig |
| future            | werden + infinitive     | Ich werde lernen.                  |
| passive           | werden + Partizip II    | Der Brief wird geschrieben.        |
| conditional       | würde + infinitive      | Ich würde lernen.                  |
